# Simple Banking System

A command-line banking application built in Python to practice Object-Oriented Programming (OOP).

## Features

- Create bank accounts
- Deposit money
- Withdraw money
- Transfer money between accounts
- View account information
- View transaction history
- List all accounts

## Concepts Practiced

This project was built primarily to learn and practice OOP concepts:

- Classes and objects
- Constructors (`__init__`)
- Instance attributes
- Instance methods
- Encapsulation
- Object references
- Dictionaries storing objects
- Exception handling
- Input validation
- Composition (`Bank` managing multiple `Account` objects)


```

## What I Learned

This project helped me understand:

- Why classes are useful for organizing related data and behavior
- How objects persist and maintain state
- How dictionaries can store objects directly
- That variables can reference the same object in memory
- Why modifying an object through one reference affects the same object everywhere else
- How multiple classes can work together to build a larger application
